Difruhe is a minimal structural language for cross-model coordination.
It makes progression append-only, referenceable, and mutually verifiable.

Difruhe is not a tool.
Difruhe is not a platform.
Difruhe is a structural foundation.

## What This Release Contains

1. Constitution v0.1  
Defines foundational axioms and structural laws:
- Append-only difference  
- Referenceable progression  
- Event-driven state  
- Canonical identity  
- Evidence-bound completion  
- Shared structural verdict  

2. Reference Canon v0.1  
Defines what counts as a valid reference:
- Must be resolvable  
- Must be verifiable  
- Must not drift  

No verifiable reference → no DONE.

3. Rule Canon v0.1  
Rules are versioned and fingerprinted.
No silent upgrades.
Completion is always relative to a specific rule version.
