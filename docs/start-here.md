# Start Here

Difruhe is not a tool.
Difruhe is not a platform.

Difruhe is a structural language for distinguishing what is merely proposed from what is valid, referenceable, and allowed to become real.

It exists for a specific boundary:

the boundary between AI intent and real-world action.

## Core claims

- Candidate is not action.
- State must come before intent.
- Admission must come before execution.
- Container validity matters.
- Execution success does not equal legitimacy.

## Versioning

- Latest language: v0.2
