# Round 1 Distribution Copy (ZH)

## 中文主引导

Difruhe 这次先不讲“怎么把 agent 做得更聪明”。

它先抢一个更底层的问题位：

**AI 意图与现实动作之间，缺了一层治理。**

当 AI 从生成语言走向触发现实动作时，
candidate output、valid state、admissible execution、legitimate action，
就不再是一回事。

Difruhe 现在先钉 5 个判断：

- candidate is not action
- state before intent
- admission before execution
- container validity matters
- execution success does not equal legitimacy
