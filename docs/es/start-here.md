# Empieza aquí

Difruhe no es una herramienta.
Difruhe no es una plataforma.

Difruhe es un lenguaje estructural para distinguir lo que todavía es solo una propuesta de lo que ya es válido, referenciable y permitido para volverse real.

Aborda un límite muy concreto:

el límite entre la intención de la IA y la acción en el mundo real.

## Juicios centrales

- Candidate is not action.
- State must come before intent.
- Admission must come before execution.
- Container validity matters.
- Execution success does not equal legitimacy.
